# Gala Kokoreç Eminönü - Web Sitesi

Gala Kokoreç'in resmi web sitesi. Eminönü Kutucu Sokak No:21'de 1970'den bu yana aynı adreste hizmet vermektedir.

## Canlı Site

- **GitHub**: https://github.com/botfusions/gala-kokorec-eminonu
- **Netlify**: Otomatik deploy (GitHub push sonrası)

## Teknolojiler

- Tek sayfa statik HTML5
- CSS Custom Properties (dark theme)
- GSAP + ScrollTrigger (animasyonlar)
- Lenis (smooth scroll)
- Lucide Icons
- Google Fonts (Space Grotesk + Inter)

## Site Bölümleri

| Bölüm | Açıklama |
|-------|----------|
| Hero | Video döngüsü arka plan + istatistikler |
| Menü | 6 ürün kartı (fiyat gösterilmiyor) |
| Midye Dolma İnfografik | 4 özellik kartı + ürün görseli |
| Hakkımızda | 19 yıldır aynı adreste vurgusu |
| Footer | Logo + sosyal medya linkleri |
